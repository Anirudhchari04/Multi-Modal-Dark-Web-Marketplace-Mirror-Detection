Expected Output: JSON with detection results and confidence score

================================================================================
USAGE EXAMPLES
================================================================================

BASIC DETECTION
$ python infer.py http://marketplace1.onion http://marketplace2.onion

WITH PGP FINGERPRINTS
$ python infer.py http://m1.onion http://m2.onion \
  --pgp-old "1234567890ABCDEF1234567890ABCDEF12345678" \
  --pgp-new "1234567890ABCDEF1234567890ABCDEF12345678"

WITH CUSTOM MODEL
$ python infer.py http://m1.onion http://m2.onion --model models/my_model.pkl

SAVE RESULTS
$ python infer.py http://m1.onion http://m2.onion --output result.json

TRAINING WITH CUSTOM DATA
$ python train.py --data training_data.csv --output models/my_model.pkl

RUN ALL EXAMPLES
$ python examples.py

PYTHON API USAGE
```python
from main import MarketplaceMirrorDetector
detector = MarketplaceMirrorDetector("models/mirror_detector.pkl")
result = detector.detect_mirror("http://m1.onion", "http://m2.onion")
print(result)
```
